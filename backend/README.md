The backend will handle:
Managing clients (buyers and sellers)
Managing property listings
Tracking deal stages (Lead → Negotiation → Closed)
Basic CRUD operations for all main entities
Storing data in a PostgreSQL database
For the MVP version, the focus will be on implementing the core functionality first, while keeping the system simple and easy to extend in the future.
Technologies

The backend will use the following technologies:
Java
Spring Boot
Spring Data JPA
PostgreSQL
REST API
Maven
The API will allow the system to create, read, update, and delete data related to clients, properties, and deals.
